# David R. Maestas, Jr., PhD - Personal Website

Static academic personal website built with plain HTML, CSS, and vanilla JavaScript.

This package contains only the deployable website files:
- index.html
- research.html
- publications.html
- cv.html
- contact.html
- style.css
- script.js
- assets/

To publish later with GitHub Pages, use this as the contents of a repository such as `david-maestas.github.io`.
